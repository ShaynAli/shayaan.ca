# Agents Guide

Write the minimal, most simple solution.

Prefer single quotes ('). If different quote types are necessary use single quotes as the outside quotes.

In Vue files, using SFC, keep the script setup at the top and style scoped at the bottom, if they are present.

For comments use complete sentences, including periods, to describe non-obvious behaviors.

Imports should be ordered such that imports that are most likely to fail, e.g. external imports, are at the top.
